ORGANIZATION (the OSI + telecom-stack layout)
======================================================================

The spec library is organized in layers, climbing the stack:

  0_Security                    Security specs across all layers (incl. Wireless RFCs, Compliance Standards)
  01_Layer2_DataLink            L2 / data link
  02_Layer3_Network             L3 / network
  03_Layer4_Transport           L4 / transport
  04_Routing                    Routing protocols
  05_First_Hop_Redundancy       HSRP, VRRP, GLBP
  06_BFD                        Bidirectional Forwarding Detection
  07_MPLS_Advanced              MPLS
  08_Segment_Routing            SR / SR-MPLS / SRv6
  09_QoS                        Quality of Service
  10_Multicast                  Multicast
  11_VPN_Tunneling              VPN
  12_Tunneling                  Generic tunneling
  13_NAT                        Network Address Translation
  14_NAT_Traversal              NAT traversal (STUN/TURN/ICE-adjacent)

  16_Virtualization             Network virtualization
  17_SDN                        Software-Defined Networking

  19_DNS_Extended               DNS

HOW TO USE THIS LIBRARY WHEN BUILDING TECH-CO
======================================================================

For each Tech-Co component you build or graft in, open the matching spec
folder first and let the spec drive the implementation.

  Building AMF       -> 45_Telecom_5G_NFs/AMF/ + 40_Telecom_3GPP/NGAP/ + 40_Telecom_3GPP/NAS/
  Building SMF       -> 45_Telecom_5G_NFs/SMF/ + 40_Telecom_3GPP/SBA_HTTP2/ + N11, N4, N7, N10
  Building UPF       -> 45_Telecom_5G_NFs/UPF/ + 40_Telecom_3GPP/GTP-U/ + N3, N4, N6, N9
  Building NSSF      -> 45_Telecom_5G_NFs/NSSF/ + N22
  Wiring IMS         -> 42_Telecom_IMS/* + 48_Telecom_Codecs/
  Wiring SMO / O2    -> 53_O-RAN/* + ETSI O-RAN TS docs
  Wiring OSS/BSS     -> 49_TMForum_Standards/* (start with TMF620, 622, 633, 638, 641, 678)
  Wiring edge / MEC  -> 52_ETSI/MEC/ + 43_Telecom_5G_Interfaces/N6/
  Wiring NFV layer   -> 52_ETSI/NFV/
  Wiring SDN         -> 17_SDN/ + 28_Network_Management/
  Wiring timing      -> 29_Timing_Sync/
  Wiring compliance  -> 47_Telecom_Security/ + 0_Security/21_Compliance_Standards/

This is the standards backbone Tech-Co runs on. Every implementation
choice should be traceable to a spec in this library.
